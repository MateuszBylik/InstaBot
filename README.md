# InstaBot
Application for solving _Instaling_

Requires Node.js (22+)

# Installation
1. Download and install Node.js (22+) https://nodejs.org/en/
2. Clone the repository:
```
git clone https://github.com/MateuszBylik/InstaBot.git
```
3. Install all dependecies:
```
npm install
```
5. To run, enter:
```
npm run start
```
